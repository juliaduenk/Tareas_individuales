border

The border property in CSS is used to define the border around an element. It combines the width, style, and color of the border into a single declaration. For example, border: 2px solid #000; sets a border that is 2 pixels wide, solid in style, and black in color.

background-image

In CSS, the background-image property is used to specify an image that will be used as the background of an element. This property allows you to add visual interest and enhance the design of your webpage by placing an image behind the content. For example, background-image: url('example.jpg'); would set the image named 'example.jpg' as the background of the element. You can also use gradients or other image formats as background images.

color

In CSS, the color property is used to specify the color of the text content within an element. By setting this property, you can change the appearance of text to match the design of your webpage. For instance, color: red; would make the text inside the element appear in red color. You can use various color formats such as named colors, hexadecimal, RGB, or HSL values to define the color.

font

In CSS, the font property serves as a shorthand for setting various aspects of text formatting including font-style, font-variant, font-weight, font-size, line-height, and font-family. This shorthand provides a convenient way to define multiple font properties in a single declaration. For example, font: italic small-caps bold 16px/1.5 Arial, sans-serif; sets the font style to italic, enables small capitals, makes the text bold, sets the font size to 16 pixels, sets the line height to 1.5 times the font size, and specifies the font family as Arial or a sans-serif fallback.

font-family

In CSS, the font-family property is used to specify the font family or typeface for the text content within an element. It allows you to control which font or group of fonts will be used to display the text. For example, font-family: "Arial", sans-serif; would set the font family to Arial, and if Arial is not available, it will fall back to a generic sans-serif font. You can specify multiple font families as fallback options to ensure compatibility across different systems.

font-style, font-weight

In CSS, the font-style property is used to specify the style of the font, such as italic or normal, while the font-weight property determines the thickness or boldness of the font. These properties allow you to customize the appearance of text by controlling its style and weight. For example, font-style: italic; would make the text italic, and font-weight: bold; would make the text bold. You can use these properties individually or combine them to achieve the desired text appearance.

width, height

In CSS, the width property is used to set the width of an element, while the height property is used to set its height. These properties allow you to control the dimensions of elements on a webpage. For example, width: 200px; would set the width of the element to 200 pixels, and height: 100px; would set its height to 100 pixels. You can use various units such as pixels, percentages, or viewport units to specify the dimensions.

list-style

In CSS, the list-style property is used to set the style of the list marker for a list item, such as a bullet or a number. This property provides control over the appearance of lists, allowing you to customize the marker type, position, and image. For example, list-style: square inside url('marker.png'); would set the list marker to a square shape, inside the list item, and use an image named 'marker.png' as the marker. You can also specify separate properties like list-style-type, list-style-position, and list-style-image for more granular control over the list marker.

margin

In CSS, the margin property is used to set the space around an element, creating a gap between the element and its neighboring elements. This property allows you to control the spacing between elements in your layout. For example, margin: 10px; would set a margin of 10 pixels on all sides of the element, while margin: 10px 20px; would set a margin of 10 pixels on the top and bottom, and 20 pixels on the left and right. You can also specify individual margin values for each side using margin-top, margin-right, margin-bottom, and margin-left properties for more precise control.

padding

In CSS, the padding property is used to set the spacing inside an element, creating space between the element's content and its border. This property allows you to control the amount of space inside an element, providing visual separation between the content and the border. For example, padding: 20px; would set a padding of 20 pixels on all sides of the element, while padding: 10px 20px; would set a padding of 10 pixels on the top and bottom, and 20 pixels on the left and right. You can also specify individual padding values for each side using padding-top, padding-right, padding-bottom, and padding-left properties for more precise control.

overflow

In CSS, the overflow property controls how content that overflows the boundaries of an element's box is handled. It specifies whether the content will be clipped, displayed with scrollbars, or completely hidden if it exceeds the element's dimensions. For example, overflow: auto; will add scrollbars only when needed, overflow: hidden; will hide any overflowing content, and overflow: visible; will allow content to overflow its container without clipping or scrollbars. You can also use overflow-x and overflow-y to control overflow behavior separately along the horizontal and vertical axes.

display

In CSS, the display property determines the type of box used for an HTML element and how it will be rendered on the webpage. It controls whether an element is treated as a block-level or inline-level element, or if it's taken out of the normal document flow. For example, display: block; will make the element a block-level element, display: inline; will make it an inline-level element, and display: none; will hide the element completely from the page layout. There are also other values such as inline-block, flex, and grid that offer more advanced layout options.

position

In CSS, the position property determines the type of positioning used for an element within its parent container. It controls whether an element is positioned statically, relative to its normal position in the document flow, or if it's taken out of the normal flow and positioned absolutely, relative to its containing block. For example, position: static; will position the element according to the normal flow of the document, position: relative; will position it relative to its normal position, and position: absolute; will position it relative to its closest positioned ancestor. Additionally, position: fixed; will position the element relative to the viewport, and position: sticky; will keep the element positioned based on the user's scroll position. These values provide flexibility in creating complex layouts and effects.

filter

In CSS, the filter property allows you to apply graphical effects to elements, such as blurring, adjusting brightness, contrast, saturation, or hue. It provides a way to manipulate the appearance of elements without modifying their actual content or structure. For example, filter: blur(5px); would apply a blur effect to the element, filter: brightness(150%); would increase its brightness, and filter: grayscale(50%);
