a- Hyperlink

Hyperlinks are essential elements in web development. They allow you to create connections between different web pages or resources. Here's a breakdown:

  • <a> tag: The anchor tag is used to define hyperlinks.
  • href attribute: This attribute specifies the URL (Uniform Resource Locator) to which the link points.
  • Visible text: The text that users see on the page (e.g., " Mejores Momentos | Gran Premio de Hungría 2021").

For example: Mejores Momentos | Gran Premio de Hungría 2021

p- Paragraph

The <p> tag defines a paragraph in HTML.

For example:
Hello, there was an eclipse today

It can also be aligned:

This text is aligned to the left

b

The <b> tag is used to make text **bold** for visual styling purposes. It doesn't convey any additional importance or semantic meaning.

For example:
This is normal text - and this is bold text.

Headers (h1, h2, ..., h6)

Header tags are used to define the hierarchical structure of a web page. Here are some key points:

  • <h1>: The most important header. Use it once per page to represent the main topic.
  • <h2> to <h6>: Subsequent headers. Use them to organize content hierarchically.
  • Level progression: Avoid skipping levels (e.g., don't jump from h2 to h6).

For example:

Header 1

Header 2

Header 3

Header 4

Header 5
Header 6
